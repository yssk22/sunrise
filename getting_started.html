Getting Started¶

Requirements¶

  • node.js v0.3.x or later
  • npm
  • couchapp v0.7.x or later

Install¶

$ npm install sunrise

Craete a new site¶

$ sunrise:create my_site
$ cd my_site
$ sunrise:start .
